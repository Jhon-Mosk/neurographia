Проект для более удобных занятий нейрографией.

Написан на node.js. В качестве базы используется sqlite.

В базе хранятся пары выражений на английском и русском Так же у них есть уровень сложности согласно Европейской классификации сложности английского языка (A1/A2/B1/B2/C1/C2). И статус, выполнено/не выполнено.

При запуске программы в терминал выводиться фраза на русском. При нажатии на Enter выводится перевод на английском. И вопрос: Изучено? На вопрос нужно ответить yes/y или no/n В базе для этой пары меняется статус на выполнено/не выполнено, в зависимости от ответа. И выводится следующая фраза.

Фразы берутся из базы самой низкой сложности со статусом не выполнено.
